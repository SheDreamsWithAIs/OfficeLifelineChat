# Compliance Guidelines

## GDPR Compliance
We are committed to GDPR compliance. Our practices include:
- Right to access: Users can request their data at any time
- Right to erasure: Users can request deletion of their data
- Data portability: Users can export their data
- Privacy by design: Data minimization and purpose limitation
- Breach notification: We notify users within 72 hours of any data breach

## SOC 2 Type II
We maintain SOC 2 Type II certification, ensuring:
- Security controls for data protection
- Availability of systems and services
- Processing integrity
- Confidentiality safeguards
- Privacy protection

## Data Processing Agreements
Enterprise customers can request a Data Processing Agreement (DPA) outlining:
- Data processing scope and purpose
- Security measures
- Data retention policies
- Sub-processor list

## Audit Logs
We maintain comprehensive audit logs for:
- User access and authentication
- Data modifications
- API access
- Administrative actions

Audit logs are retained for 7 years for compliance purposes.

## Data Residency
- US data centers: US customers' data stored in US
- EU data centers: EU customers' data stored in EU (GDPR compliant)
- Data residency options available for Enterprise customers

## Certifications
- SOC 2 Type II
- ISO 27001 (in progress)
- HIPAA compliance available for Enterprise Healthcare customers
